## КЛАССИФИКАЦИЯ — 4 ГЛАВНЫЕ КАТЕГОРИИ

### 🏠 RESIDENTIAL (самое частое)

```
1. Broken Window / Glass Replacement
   → Тип рамы: Wood / Aluminum / Vinyl / Steel
   → Стекло:
      Clear (по умолчанию)
      Tinted: Bronze / Gray / Dark Gray
      Textured (privacy): SprayLite / Rain / Reeded / P516 / Acid Etch
   → Толщина: 1/8" / 3/16" / 1/4" / 3/8" / 1/2"
   → Закалка: Annealed / Tempered (для дверей, ванных, рядом с полом)
   
2. Double Pane IGU (стеклопакет)
   → Пакет:
      Clear/Clear (стандарт)
      Clear/Low-E Advantage
      Clear/Solarban 70
      Tinted/Clear (Bronze или Gray снаружи)
   → Спейсер: Silver / Bronze / Black
   → Газ: Air / Argon
   → Закалка: Annealed / Tempered
   → Решётки (grids): Нет / SDL / GBG

3. Foggy IGU (туманный стеклопакет)
   → Только замена вкладыша (рама остаётся)
   → Те же опции IGU
   → Материал рамы (влияет на трудозатраты)
```

### 🏢 COMMERCIAL

```
4. Storefront Glass
   → Тип:
      Single Pane Tempered (чаще всего)
      IGU Storefront (энергоэффективный)
   → Стекло:
      Clear Tempered (обязательно)
      Tinted Tempered: Bronze / Gray / Reflective
      Reflective: Mirror Pane / Solar Cool Gray / Solar Cool Bronze
   → Толщина: 1/4" / 3/8" / 1/2"
   → Reflective только 1/4" (Liberty catalog)

5. Office Partition / Interior Glass
   → Clear Tempered (чаще)
   → Textured / Acid Etch (для приватности)
   → Annealed допустим (интерьер)

6. Commercial Door Glass
   → Tempered обязательно
   → Wire glass (fire rated) — опционально
```

### 🚿 SHOWER & BATH

```
7. Shower Door / Enclosure
   → Конфигурация:
      Single Door (swing)
      Double Door (French)
      Sliding Bypass
      Fixed Panel + Door
   → Стекло (всегда tempered):
      Clear (стандарт)
      Textured/Obscure: Rain / Acid Etch / SprayLite
      Low Iron (Starphire) — ультра-прозрачное
   → Толщина: 3/8" / 1/2" (frameless) / 1/4" (framed)
   → Hardware (от Liberty):
      Hinges: Chrome / Brushed Nickel / Matte Black / Bronze
      Clamps: те же цвета
      Handle style: standoff / ladder / towel bar
   → Покрытие: ClearShield / EnduroShield (water repellent)

8. Shower Enclosure (полная кабина)
   → То же + боковые панели
   → Размеры нестандартные → custom quote
```

### ✨ SPECIALTY

```
9. Table Top Glass
   → Форма: Rectangle / Round / Custom
   → Толщина: 1/4" / 3/8" / 1/2"
   → Тип: Clear / Low Iron (без зелёного оттенка)
   → Edge: Polished flat / Pencil / Beveled
   → Holes: центральное (umbrella) / custom
   → ВСЕГДА tempered (обязательно)

10. Custom Mirror
    → Standard Silver / Low Copper (влагостойкое)
    → Толщина: 1/8" / 3/16" / 1/4"
    → Edge: Seamed / Polished / Beveled
    → НЕ закалённое (несовместимо с покрытием серебра)

11. Sandblasted / Custom Design Glass
    → Liberty предлагает готовые дизайны (PDF каталог)
    → Матовые узоры, логотипы, паттерны
    → Применение: офисные перегородки, душевые, входные двери
    → Цена: по запросу (custom)
```

## ПРОДУКТЫ LIBERTY — КАК ИСПОЛЬЗОВАТЬ В КАЛЬКУЛЯТОРЕ

### Textured Glass (от Liberty)
```
Применение: душевые, ванные, офисные перегородки, входные группы
Варианты (стандартные толщины):
  SprayLite:   3/16", 1/4"
  Rain:        1/8", 5/32", 3/16", 1/4", 3/8"   ← самый популярный
  Reeded:      5/32", 1/4"
  P516:        1/8", 5/32", 3/16", 1/4"
  Acid Etched: 1/8", 1/4", 3/8", 1/2"            ← premium

Показывать в калькуляторе как:
  "Obscure/Privacy Glass" → выпадает список паттернов
```

### Reflective Glass (от Liberty, только 1/4")
```
Варианты:
  Mirror Pane (зеркальное)
  Solar Cool Gray (тонировка серая + отражение)
  Solar Cool Bronze (тонировка бронза + отражение)

Применение: storefront фасады, коммерческие здания
Показывать: только в Commercial категории
```

### Tinted Glass (от Liberty)
```
Bronze:    1/8", 3/16", 1/4", 3/8", 1/2"
Gray:      1/8", 3/16", 1/4", 3/8", 1/2"
Dark Gray: 1/8", 1/4"

Показывать: в Residential (window tint option) + Commercial
```

### Fabrications (от Liberty)
```
Cut-outs: отверстия, вырезы (для розеток, труб, кронштейнов)
Sandblasting: матирование + дизайны (PDF каталог есть)
Edgework: полировка краёв
→ Это ADD-ONS (+$X) к любому продукту
```
